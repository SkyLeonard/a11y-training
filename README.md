30 Days of Accessibility
===

30-day accessibility email course that I produced a few years ago and made it into a website.

This is broken into single-topic pages with an overview of the requirement, how to test for it, guidance on how to ensure the criteria is met, and tools.

This should be the starting point to read through in order.

## Local build

1. `pip install -r requirements.txt`
1. `mkdocs build --strict`
1. `mkdocs serve -a 127.0.0.1:8001`
1. `open http://127.0.0.1:8001`
